# AI_LLM_PD1
Es izmantoju lokālu vienkāršotu valodas modeli, kas balstīts uz atslēgvārdiem un noteikumiem, jo nesanāca ar citiem LLM.
Tomēr, modelis analizē tekstu un aizvieto sarežģītus terminus ar vienkāršākiem skaidrojumiem.
Programmu varētu uzlabot, pievienojot vairāk terminu, sinonīmus vai teksta analīzi.
Nakotnē pamēģināšu arī citus LLM.
